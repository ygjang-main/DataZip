# Filezill 설치(Windows, macOS)     
- 오픈 소스 FTP 클라이언트 프로그램     

1.  



1. FileZilla 설치(macOS 기준)
(1) 다운로드 URL : https://filezilla-project.org/download.php?platform=macos-arm64

(2) FileZilla_~~~ .app.tar.bz2 파일을 클릭하여 압축해제
(3) FileZilla.app 실행
(4) [열기]
(5) 네트워크 볼륨의 파일에 접근 : 허용
(6) 접속 및 테스트 
1) 사이트 관리자 클릭

2) 새 사이트(N)

3) 연결 설정
- 일반 탭
- 프로토콜 : SFTP - SSH File Transfer Protocol
- 호스트 : 원격시 서버 IP주소 등 입력
- 포트 : 기본 포트(22) 또는 SSH 설정한 포트 입력
- 로그온 유형 : 일반
- 사용자 : 원격지 서버 계정
- 비밀번호 : 원격지 서버 비밀번호
4) 테스트 완료
